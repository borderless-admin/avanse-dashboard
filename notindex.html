⚙️ Google Sheet Connection
📋 One-time setup — takes 3 minutes:

Step 1: Open your Google Sheet → click Extensions → Apps Script

Step 2: Delete any existing code and paste this entire script:
function doGet() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheets()[0]; const data = sheet.getDataRange().getValues(); const headers = data[0]; const rows = data.slice(1).map(row => { const obj = {}; headers.forEach((h, i) => obj[h] = row[i]); return obj; }); return ContentService .createTextOutput(JSON.stringify(rows)) .setMimeType(ContentService.MimeType.JSON); }
Step 3: Click Deploy → New deployment
  • Type: Web app
  • Execute as: Me
  • Who has access: Anyone
  • Click Deploy → copy the Web App URL

Step 4: Paste that URL in the box above → click Connect & Load Live Data ✓

⚡ This is a one-time setup. After this, just click Sync to get live data anytime.
